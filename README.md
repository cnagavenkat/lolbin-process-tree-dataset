# LOLBin Process Tree Dataset — Sample

This repository contains a representative sample of the labeled process tree 
dataset introduced in:

> "Beyond the Binary: Detecting the Malicious Abuse of Living-Off-The-Land 
> Techniques using Graph Neural Networks" — Cyber Security and Applications (under review), 2026.


Each file is a JSON process tree with nodes representing process invocations 
and edges representing parent-child spawn relationships.

## Full Dataset

The complete dataset of 7,008 labeled process trees will be released here 
upon acceptance of the paper.
